# Карточки по ИИС — GitHub Pages

Это готовый статический сайт для GitHub Pages.

## Как загрузить
1. Создай новый репозиторий на GitHub
2. Загрузи в него файл `index.html`
3. Открой Settings → Pages
4. В разделе Build and deployment выбери:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
5. Сохрани настройки

Через короткое время сайт появится на GitHub Pages.

## Возможности
- список всех вопросов
- поиск
- переворот карточки
- отметка «прочитано»
- отметка «сложный»
- сохранение прогресса в браузере
